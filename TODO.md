## Features

1. Authentication

2. Facility Locator
    - Locator
    - Home Pickup 
    - Feedback System
    - Monitor Different recycling stage

3. Reward / Credit System
    - Scratch card and normal rewards
    - Shop for redeeming rewards
    - Gamification (optional)

4. Educational Popups (Can use AI)

- Notification

The facility will pickup from that location if a certain amount of e waste or certain price limit is met in that area, if limit is not met then facility will send truck to collect.



## Use Cases

- User
    - Login/Signup (User)
    - Enter location
    - select item
    - select facility
    - get notification of scheduled pickup
    - get credit points, redeem reward
    - browse educational content
    - track the item status
    - give feedback.

- Facilities
    - Login/Register (Facility)
    - Listing acceptable e-waste products
    - View user request
    - approve request
    - schedule pickup
    - tracking pickup/item
    - rewarding credit points to the user
    - listing recycled products in the shop.

- Admin
    - Login/Signup
    - manage accounts
    - add/remove accounts
    - manage third-party services
    - promote educational contents
    - manage the shop for recycled products
    - give notifications on upcoming events
    - reviewing and managing feedback.
